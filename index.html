The name is ironic.
The robots are not.

We're a student robotics team in the FIRST Tech Challenge: an international competition where high schoolers design, build, and program 18-inch robots to play a brand-new game every season.

We do the engineering, the fundraising, the outreach, and the fire-watch. The name sets expectations on the floor so the robot can clear them. Judges call it memorable; we call it insurance.

Full team story

This season's machine

Codename: Bagel Destroyer. Built for this season's game, named by committee, feared by baked goods.

Every season starts the same way: a new game drops in September and we get about twelve weeks to design, prototype, and program a machine that can play it. Bagel Destroyer is the current answer. It drives, it scores, and it has only briefly smelled like burning.

The full CAD, code, and build notes live on the Resources page. We publish everything, including the mistakes. Especially the mistakes.
